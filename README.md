# DuffPac-SFML-Game
Game produced in 2nd term at NTU, Insprited by PacMan. Written in C++ and the SFML library.

# Installation 
Release Folder is ready-to-go. you can rebuild yourself but will need to retarget to v141 MSBuild tools
